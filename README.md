# Personal Portfolio Website

This is my personal portfolio website, built to present my profile, skills, projects, internship experience, technical interests, and contact details in a clean and modern way.

I created this website as a personal space on the web where I can showcase my work, highlight my background as a B.Tech graduate, and keep updating my professional journey over time.

## About The Website

The portfolio is designed with a modern dark theme, elegant typography, responsive layout, and subtle scroll animations. It is built using plain HTML, CSS, and JavaScript to keep the structure simple, lightweight, and easy to maintain.

## Sections Included

- Home
- About
- Skills
- Projects
- Experience
- Blog
- Contact

## Features

- Responsive design for desktop and mobile
- Dark theme with a clean modern interface
- Smooth scrolling navigation
- Scroll-based reveal animations
- Hero section with profile image and social links
- Categorized skills section
- Project showcase cards
- Internship section with organization logo
- Blog section layout for technical writing
- Contact section with social and communication links

## Tech Stack

- JavaScript

- Google Fonts

## Running Locally

To run this project locally, I use:

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Deployment

This portfolio is ready to be deployed on Netlify as a static website.
